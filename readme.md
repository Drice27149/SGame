### 射击战场

使用 UE4 引擎制作的第三人称射击游戏

### 玩法简介

* 第三人称射击游戏

  玩家可进行移动, 瞄准, 射击

  可以拾取和切换不同的枪械

* 敌人为 AI 控制的角色

  不同的敌人会有不同的行为和特性

  敌人和玩家均有血量设定, 受到攻击后血量减少, 玩家血量小于 0 后游戏失败, 敌人血量小于 0 后被击败

* 击败关卡中所有敌人为通关

  关卡中会随机刷新 buff, 玩家拾取获得血量/子弹增益
